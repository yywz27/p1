# p1-bill 消费账单数据分析实战项目
## 项目介绍
这是我的第一个数据分析入门练手项目，独立完成全业务流程：
模拟消费账单原始数据 → Python Pandas清洗脏数据 → MySQL建表与多维度聚合统计 → Excel数据可视化 → 输出完整分析总结报告。

## 项目文件结构
p1-bill/
├── python_script    # 数据清洗Python源码
├── raw_data         # 原始账单文件、清洗完成后的数据集
├── sql_backup       # MySQL数据库备份文件、查询脚本
└── 消费账单数据分析实践报告.docx  # 完整项目复盘文档

## 使用技术
Python（Pandas）、MySQL、Excel、数据可视化
